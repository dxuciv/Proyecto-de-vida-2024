# Proyecto de Vida - Seminario 2024

## Descripción

Este proyecto de vida tiene como objetivo presentar una línea de tiempo detallada de los hitos significativos en mi vida
 hasta la fecha, destacando logros personales, experiencias y momentos clave.

El proyecto está desarrollado utilizando HTML, CSS y JavaScript para crear una experiencia interactiva y visualmente
 atractiva que ilustre la evolución de mi vida a lo largo de los años.


## Tecnologías Utilizadas

HTML5: Estructura general del contenido
CSS3: Estilos para la presentación
JavaScript: Interactividad y lógica

## Instalación

Para ver el proyecto en funcionamiento localmente, sigue estos pasos:

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/dxuciv/PDV-2024.git
   ```

2. **Navega al directorio del proyecto**:
   ```bash
   cd tu_repositorio
   ```

3. **Abre el archivo `index.html` en tu navegador**:
   ```bash
   open index.html
   ```
   o simplemente arrastra y suelta el archivo `index.html` en una ventana de tu navegador.

## Uso

Al abrir el archivo `index.html`, verás una línea de tiempo que muestra los hitos de vida ordenados cronológicamente.
A medida que te desplazas por la página, los elementos se animan y se vuelven visibles.

## Ejemplos

Aquí tienes algunos ejemplos de los hitos incluidos en la línea de tiempo:

- **1 Año**: Comencé a dar mis primeros pasos y a decir mis primeras palabras.
- **4 Años**: Ingresé al jardín de infancia 'Mi Mundo' y recibí mi primera bicicleta.
- **16 Años**: Compré mi primera motocicleta y participé en mi primera carrera.
